Mu-tune

VR Game for Mobile & Oculus Go

Jul 26, 2018 #logo #3D #ui #movie #planning
制作期間

1.5ヶ月

担当者数

開発者1名 / デザイナー1名

作業内容
  • 背景・小道具 : デザインからモデリング・テクスチャ制作、Prefab化、Unityシーンビルド
  • UI : デザイン、Unity上でのレイアウトまでを担当
  • その他ゲーム内作業 : 企画・提案、エフェクト、シェーダー修正、簡易C#スクリプト、ゲーム内テキスト
  • ゲーム外制作物 : ロゴ、ストア画像アセット、プロモーション動画
詳細
  • Android端末とOculus Go用に開発したリズムゲームです。、Youtubeから動画を引用し、譜面を自動生成する機能（特許出願中 特願2018-175664号）を持っています。
  • 限られた開発期間の中で「音ゲーとしての気持ち良さ」の調整に多くの時間をとるため、画面内のほとんどの要素を画像(skybox)で構成し、エフェクトや操作感の調整を重点的におこないました。
